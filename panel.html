🛠️ Панель КП

Доступ только для owner / curator / gm / mod. Роли берутся из таблицы profiles.
Проверяю сессию...

⛔ Доступ ограничен

1) Войди через Google.
2) Затем в Supabase назначь роль в таблице profiles.
3) После обнови страницу.
Подсказка по структуре таблиц:
profiles: id (uuid, = auth.users.id), role (text), display_name (text), discord_id (text).
characters: id (uuid), owner_id (uuid), name (text), imprint (text), faction (text), bio (text), approved (bool), created_at (timestamptz).

🔐 Как выдавать роли

Роли выдаются в таблице profiles (колонка role).
Примеры: owner, curator, gm, mod, user.
Для твоей команды можно заранее создать записи в profiles с discord_id:
